Trypto


Project Description :

Trypto is a web 3.0 Blockchain App. It allows the users to connect their Metamask Wallet and send their Ethereum to other cryptocurrency wallets. These transactions are stored in the blockchain networks with smart contracts. They can view the transaction records after transaction is done. Furthermore, they can also view the cryptocurrency markets in the website.

This project was the blockchain application learning project. The challenges of the project are not familiar with the new trends of application, espiecially for the solidity which is used to write the smart contract and get the transaction records from the blockchain network. With this, I had watched many videos on YouTube to figure out the ways of it. This project is quite interesting as it is something new for me. At the end of the project, I had strengthen my skill in building a frontend with React JS and the learnt the concept of building a blockchain app.

Tech Stack:

- Frontend : Vite React JS Tailwind CSS
- Smart Contracts : Solidity
- Cryptocurrency Trackers : CoinGecko API
